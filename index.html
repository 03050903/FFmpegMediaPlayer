FFmpegMediaPlayer is a reimplementation of Android's MediaPlayer class. The FFmpegMediaPlayer class provides a unified interface for playing audio files and streams.
You can browse the javadoc for more information, including example usages.

The main advantages of this library vs. MediaPlayer are:
  • Support for API 7+ (Not just API level 10+ like MediaMetadataRetriever)
  • URL support (Unlike MediaMetadataRetriever, see: http://code.google.com/p/android/issues/detail?id=35794)
  • Support for additional protocols including file, http, https, mms and mmsh
  • Support for additional audio and video formats including aac, acc+, flac, mp2, mp3, mp4, ogg, 3gp and more!
  • Support for ICY Metadata (SHOUTcast metadata)
